# Hosting Test - GitHub Pages with Git LFS

This is a test of hosting a Hugo site on GitHub Pages with Git LFS to store media.

To test locally you must install Git LFS.

```text
GIT_CLONE_PROTECTION_ACTIVE=false git clone https://github.com/jmooring/hosting-github-pages-git-lfs
cd hosting-github-pages-git-lfs
hugo server
```
